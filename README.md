   ->Travel Website UI

A modern, responsive, and immersive landing page for a travel agency.

   ->Description

This project is a high-quality User Interface (UI) designed to showcase dream travel destinations. It features a clean layout, beautiful imagery, and a smooth user experience across all devices.

   ->Features

Fully Responsive Design: Optimized for mobile, tablet, and desktop screens.

Dynamic Gallery: Beautiful presentation of destinations like Istanbul, Marrakech, and Kyoto.

Interactive Navigation: Smooth scrolling and intuitive menu.

Clean Code: Semantic HTML5 and modern CSS3 techniques.

   ->Technologies Used

HTML5: Structure and content.

CSS3: Styling, Flexbox, Grid, and animations.

JavaScript: (Optional) For interactive elements and UI logic.
